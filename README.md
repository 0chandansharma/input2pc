# input2pc
Android application works as touchpad
There are 3 modes to use the application- Mouse, Presentation and Keyboard.

Development Toolkit:
•	SDK (Software Development Kit)
•	Eclipse
•	Emulator
•	ADT Plugin
•	Bluecove library
